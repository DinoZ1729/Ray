## Welcome

Welcome to Ray. A C++ program that can do Raytracing in the console. This is pure bit-beauty.

### Linux compile

In order to compile and run on a Linux/GNU distro, you will need to run the following commands:

* $ **g++ *.cpp -lm**
* $ **./a.out**
